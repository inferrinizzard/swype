• Isyarat Swype

    Isyarat Swype adalah tombol pintasan pada keyboard untuk dengan cepat melakukan pekerjaan yang umum.


    Fitur ini tidak tersedia ketika layanan aksesibilitas sistem Explore-by-Touch sedang aktif.


    • Membuka Bagian Menyunting Keyboard

      Untuk membuka Menyunting Keyboard, Swype dari ke tombol Simbol (?123) pada keyboard.

    • Membuka Keyboard Angka

      Untuk membuka Keyboard angka dengan cepat, Swype dari tombol ke tombol angka 5.

    • Menyembunyikan Keyboard

      Untuk dengan mudah menyembunyikan keyboard, cukup Swype dari tombol Swype ke tombol spasi mundur.

    • Menonaktifkan Spasi Otomatis

      Nonaktifkan spasi otomatis sebelum kata berikutnya dengan men-Swype dari Tombol spasi ke Spasi mundur.

    • Menyunting Kapitalisasi Kata

      Ubah huruf besar dari kata setelah memasukannya dengan menyentuh kata dan kemudian meng-swype dari ke tombol Shift . Daftar Pilihan Kata dengan opsi kapitalisasi akan ditampilkan, Memungkinkan Anda untuk memilih huruf kecil, berhuruf Kapital, atau SEMUA KAPITAL.

    • Tanda baca

      Cara sederhana untuk memasukkan tanda baca adalah dengan meng-Swype dari tanda tanya, koma, titik, atau tanda baca lain ke tombol Spasi, alih-alih menyentuhnya.

    • Pilih Semua, Potong, Salin dan Rekat Pilih Semua: Swype dari ke 'a'
      Potong: Swype dari ke 'x'
      Salin: Swype dari ke 'c'
      Rekat: Swype dari ke 'v'
    • Pintasan aplikasi

      Google Maps: Swype dari ke 'g', lalu 'm'

    • CariSorot beberapa teks dan Swype dari ke S untuk melakukan pencarian web cepat.
    • Mengalihkan ke bahasa yang terakhir digunakan.Saat menggunakan beberapa bahasa, cara cepat untuk beralih kembali ke bahasa sebelumnya adalah dengan men-Swype dari ke tombol Spasi.
